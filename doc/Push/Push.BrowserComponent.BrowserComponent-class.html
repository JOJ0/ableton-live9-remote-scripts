Class BrowserComponent

source code

_Framework.CompoundComponent.CompoundComponent --+
                                                 |
                                                BrowserComponent

Component for controlling the Live library browser. It has 4 browsing columns that are controlled by encoders and state buttons. The contents of these lists are provided by a browser model -- see BrowserModel and derivatives.
