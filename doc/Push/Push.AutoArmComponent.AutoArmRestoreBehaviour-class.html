_Framework.ModesComponent.LatchingBehaviour --+

                                             AutoArmRestoreBehaviour

Mode button behaviour that auto-arm is enabled when the mode is activated. If it is not, then it will make the button blink and restore it in the second press.

Note that this interface is passive, you have to manually call update() to make sure the light is update when the auto-arm condition changes.
