source code

_Framework.ButtonElement.ButtonElement --+
                                         |
                                        ConfigurableButtonElement

Special button class that can be configured with custom on- and off-values.

A ConfigurableButtonElement can have states other than True or False, which can be defined by setting the 'states' property. Thus 'set_light' can take any state or skin color.
