Class Selection

source code

object --+
         |
        Selection

Object that encapsulates selection.

[jbo] The intent of this object is to grow its interface until no 'view.selected_*' access are done in any part of the script. This way, it should be easy to change our selection model such that multiple-L9C controllers can play together properly. Also, feel free to make these properties listenables when neccesary.
