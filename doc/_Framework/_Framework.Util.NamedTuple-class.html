Class NamedTuple

source code

object --+

        NamedTuple

Immutable object that acts like a dictionary whose members can also be set via attribute access. Derivatives can give and override default values in the class definition, for example:

 class MyNamedTuple(NamedTuple):
     some_value = 3

 assert MyNamedTuple == NamedTuple(some_value = 3)
